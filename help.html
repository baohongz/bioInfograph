User Guide

The web-based application bioInfograph allows user to easily arrange multiple plots in SVG, png, jpeg, or gif format exported by other tools. Each individual plot can be adjusted in size and positioned freely on the canvas. Main functions are highlighted in red callout boxes. A basic workflow is outlined by numbered text boxes.


Input of paragraphs of text by clicking on “Add Text Box” and then typing in the resizable box that was added to the canvas a moment ago. Clicking on text will fire up the formatting menu while clicking on any area outside the box will exit the menu. Formatting is always applied to selected text.


Each plot will get a button handle when it is loaded onto the canvas. A plot and the linked handle will be highlighted in yellow when hovering mouse over a handle. Dragging the handle and dropping it at desired position among these buttons will change the relative vertical stacking, also known as z-index of the plot. The bottom position represents the top layer of the stack of plots on the canvas.


Overlaying colored clustering plot and histological image to illustrate the relationship of histological features and cell clusters based on 10X Genomics spatial transcriptomics data (https://bit.ly/3F0xKWD). User can adjust the opacity of a selected plot by clicking a plot handle to select the plot and then typing a number or using the slide to change the value.
